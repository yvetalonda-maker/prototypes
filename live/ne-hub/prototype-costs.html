Amortizace setupu

Setup projektu (CLAUDE.md, design tokeny, komponentová knihovna, fonty) stál ~13 % týdenního limitu, ale je jednorázový. Každá další stránka z Figmy stojí jen ~4–5 % limitu. Čím víc stránek vznikne, tím víc se počáteční investice vyplatí.

Celkový čas session

~2,5 hodiny od prvního zadání po kompletní prototyp se 3 stránkami, interaktivitou, side panelem, dynamickými filtry, migrací ikon a publikací na GitHub Pages. Spotřebováno ~44 % týdenního limitu Claude Code (Opus). Z toho zadávání a rozhodování na straně uživatele cca 20–25 minut, zbytek je generování kódu a Figma MCP volání.
