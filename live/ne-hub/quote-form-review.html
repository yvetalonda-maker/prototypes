Komponenty — Custom vs Shared

Toggle switch

Aktualni (custom .tgl)

35x21px, teal active (#1a7282), custom CSS

Shared (.toggle)

PrimeOne toggle z components.css

Buttons

Aktualni (inline styles)

Inline styly, teal primary (#1a7282), slate secondary

Shared (.btn)

.btn .btn-primary / .btn-secondary / .btn-text z components.css

Select dropdown

Aktualni (inline styles + SVG chevron)

Custom select wrapper s SVG chevronem

Shared (.select)

.select z components.css — nativni chevron

Number input (spinner)

Aktualni (custom .num-input)

100px, custom spin buttons, inline SVG arrows

Shared (.input)

.input z components.css — nativni number stepper

Segmented control (product selector)

Aktualni (custom .sb-opt)

Custom segmented control

Shared (.tabs)

.tabs .tab z components.css — vizualne odlisne (underline vs pill)
